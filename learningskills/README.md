# learningskills

这个目录不是“原始日志堆放区”，而是“把一次次画图经验整理成可复用知识”的地方。

## 目标

把 agent 在画图过程中解决过的困难，沉淀成下次还能复用的经验。

## 建议分层

- `visio-iteration-journal/`
  - 教 agent 如何做复盘和沉淀
- `templates/`
  - 经验记录模板
- `lessons/`
  - 归纳后的经验条目

## 什么应该写进来

- 文本爆掉的根因与修复方法
- connector 过早绘制造成画面混乱的经验
- 导出 PNG 后才暴露出的排版问题
- Visio/bridge 运行时恢复策略

## 什么不应该直接写进来

- 一次运行的原始 stdout
- 含敏感信息的 token、路径或本地环境细节
- 只适用于单次任务、不可泛化的备注

经验要写成“下次还能复用”的形式，而不是流水账。
